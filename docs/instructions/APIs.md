## LLM API

Более подробно про получение и настройку API описано в .env.example python-скрипта и в документации там.

Некоторая база и выжимка:

### LM Studio

Просто и локально запустить LLM у себя на видеокарте. Будет больно, если меньше 6G VRAM. Для Mac'ов проще, можно запускать даже на 16G версиях.

### Mistral API

Самый простой API без ограничений

### Google Gemini API

Получать через Google AI Studio, для подключения и использования API понадобится EU/US proxy.
