## Предварительная подготовка

> ИНСТРУКЦИЯ ПО АКТУАЛЬНОЙ ВЕРСИИ ПОДГОТОВКИ, УСТАНОВКИ И ЗАПУСКА ЛЕЖИТ В РЕПОЗИТОРИИ

1. Комфортнее всего вам будет с установленным менеджером пакетов UV. С ним будет проще, быстрее и удобнее всего, и в нём можете установить Python 3.10.9. (более подробные инструкции должны быть в репозитории со скриптом)
2. Для качественной TTS и системы фильтрации вам понадобится Docker (желательно Docker Desktop)
3. Для STT / TTS сервисов вам понадобятся CUDA Toolkit (12.9)

Но это предварительные инструкции, актуальная версия лежит в самом репозитории!
